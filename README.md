## 部署

- 环境：
  - [Nodejs](https://nodejs.org/): 18.0.0+
- 克隆仓库：

  ```shell
  git clone https://github.com/Jursin/ACS-Web.git
  cd ACS-Web
  ```

- 相关命令：
  - 安装依赖

    ```shell
    npm install
    ```
  - 启动开发服务器

    ```shell
    npm dev
    ```
  - 构建静态文件

    ```shell
    npm run build
    ```
  - 预览网页

    ```shell
    npm preview
    ```
